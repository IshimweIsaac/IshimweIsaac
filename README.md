### Projects

**LUCAS** -- Voice-controlled AI desktop assistant
```
Stack:  Python, Gemini Live API, PyQt6, Playwright, OpenCV, FastAPI
Size:   42,724 lines, 170 files, 50 test files
```
Real-time voice-controlled AI assistant for Linux. Maintains a persistent
WebSocket connection to Google Gemini 3.1 Flash Live, streams audio and screen
video at ~1 FPS, and dispatches tool calls across 35+ action modules including
desktop automation, browser control, multi-agent swarm coding, document
generation, git/GitHub workflow automation, 18 specialized learning agents,
and a 3-tier memory system (long-term, session brief, conversation journal).
Features a PyQt6 desktop UI with 31+ windows, a Telegram bot interface, and a
hot-reloader for live code updates.
[github.com/IshimweIsaac/Lucas](https://github.com/IshimweIsaac/Lucas)

**VoiceAgent** -- AI phone receptionist for small businesses
```
Stack:  FastAPI, Twilio, Gemini Live API, Google Calendar API, SQLAlchemy
Size:   7,878 lines, 31 files, 172 tests
```
Standalone product that receives inbound phone calls via Twilio Media Streams,
processes them through Gemini Live API (Charon voice, <500ms latency), and can
book appointments via Google Calendar, answer FAQs from a knowledge base,
escalate to humans, and send SMS confirmations. Multi-tenant architecture with
a web dashboard for business configuration. Custom u-law/PCM audio resampling
pipeline with no external audio libraries.
[github.com/IshimweIsaac/VoiceAgent](https://github.com/IshimweIsaac/VoiceAgent)

A statically typed, declarative, reactive programming language designed for
